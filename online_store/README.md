# Online Store

| Endpoint | HTTP Method | Result |
| ---- | ---- | ---- |
| /store | GET | get all stores |
| /store/:name | GET | get named store |
| /store/:name/item | GET | get items from named store |
| /store | POST | create new store |
| /store/:name/item | POST | create new item |
| ... | ... | ... |
